Item Metadata in loc.gov Search Result Records

The table below displays the 216 metadata fields found from a sample of query result items across the 428 collections in loc.gov. These fields are returned in JSON query records using the loc.gov API. Additional item metadata fields may be found directly in item-level and resource-level records. For each metadata field below, one sample record was pulled from each collection on loc.gov. In 21 collections, a sample could not be found (usually because the collections contained less than 10 records). Thus, a total of 407 samples were pulled for each metadata field. Nested fields appear as in resources > 0 > url, where resources is the top-level field within the result object, 0 is the first item within the list contained in resources, and url is a field within that list item. In the case of fields nested within lists (such as this example), samples have been pulled from the first list item but not from subsequent list items (as a result, you will not see resources > 1 > url in this table). Code at https://github.com/reliztrent/metadata-sampler

For more documentation about the loc.gov API and records, see Library of Congress's About the loc.gov API. .
